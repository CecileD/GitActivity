# GitActivity
Prise en main de git
Les étapes
1-Créer un compte sur github
2-Créer un nouveau repository attaché à son compte sur github
3-Cloner le répertoire sur son poste de travail
4-Modifier le fichier readme et le convertir en fichier texte
5-Faire un commit du fichier readme.txt et un push
6-Créer les fichiers utiles au projet
7-Ajouter les fichiers nouvellement créés à l'index git
8-Mettre à jour readme.txt
9-Faire un commit et un push sur github
10-Mise à jour du fichier CommandeGit.txt
11-commit
12-Mise à jour du fichier LogGit
13-Commit et push
14-Mise à jour readme.txt
15-Commit et push finaux
